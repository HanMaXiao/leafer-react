# leafer-react Playground

Interactive playground for testing leafer-react plugin features.

## Starting the Playground

```bash
npm run dev:page
```

Then open http://localhost:5173 (or the port shown in terminal) in your browser.

## Examples

| Example | Description |
|---------|-------------|
| Basic JSX | Create Leafer graphics using JSX syntax |
| React Component | Render React UI components on Canvas |
| Interactive | Click, hover, and drag events |
| Animation | State-driven animations |

## Features

- **Hot Module Replacement**: Changes to examples apply instantly
- **Code Preview**: View source code for each example
- **Debug Mode**: Enable console logging
- **Responsive Design**: Works on different screen sizes
